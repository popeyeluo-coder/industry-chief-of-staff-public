# 行业大秘书 · Industry Chief of Staff

> 业务战略陪跑助手，从方案拆解到项目落地的完整闭环。

## 🎯 这是什么

一个 WorkBuddy / Claude / Codex 通用的 skill，让 AI 变成你的"大秘书"——不只是回答问题，而是：

- 方案审阅（红蓝对抗 / 穷举验证 / 叙事免疫）
- 竞对分析
- 项目全周期管理
- 结构化输出与汇报

行业 / OKR / 竞对 / 框架等具象信息由**使用者记忆文件**定义，skill 本身**行业无关**、通用可复用。

## 📦 如何使用

1. 把 `SKILL.md` 放到你的 AI 工具的 skills 目录
   - **WorkBuddy**：`~/.workbuddy/skills/industry-chief-of-staff/SKILL.md`
   - **Claude Code**：`~/.claude/skills/industry-chief-of-staff/SKILL.md`
2. 在对话中触发关键词：`大秘书` / `方案审阅` / `红蓝对抗` / `竞对分析` / `项目管理`
